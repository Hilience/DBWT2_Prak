Vergleich von JavaScript und C++
6 Unterschiede:
1. dynamisch typisiert vs nicht dynamisch typisiert:
let var;  // JS kann alle Typen annehmen
int var; // C++ Typ ist festgelegt

2. Rückgabetyp von Funktionen:
function xy() { ... }   // JS der Rückgabetyp muss nicht angegeben werden, auch nicht ob es einen gibt oder nicht.
void function xy() { ... }  // C++ der Rückgabetyp ist fest definiert

3. Ausführung:
// JS wird vom Browser von einer Engine direkt ausgeführt
// C++ wird compiliert zu Maschinencode und dann ausgeführt

4. Speicherverwaltung
// JS automatische Speicherverwaltung
// C++ Speicherverwaltung erfolgt größtenteils manuell

5. Anwendungsbereich
// JS fast ausschließlich für webentwicklung
// C++ so gut wie nie für webentwicklung

6. Pointer
// JS Referenzen sind versteckt, keine Pointer
// C++ Referenzen müssen selbst behandelt werden, es gibt pointer etc.

6 Gemeinsamkeiten:
1. Beides sind imperative Sprachen
2. Beides sind objektorientierte Sprachen
3. Es gibt Variablen und Funktionen
4. Es gibt die gleichen Kontrollstrukturen, if, else, switch, while, do ... while
5. Es können andere Codekonstrukte importiert werden
6. Es gibt Konstanten
